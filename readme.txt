== Installation ==

To install WooCommerce Name Your Price Event Tickets:

1. Download the extension from your WooThemes account

2. In your WordPress admin, go to Plugins > Add New and then click on the "Upload" tab

3. Click the "Choose File" button, select the zip file you just downloaded to your computer and then click "Install Now"

4. After installation has completed you can activate the plugin right away or you can activate the 'WooCommerce Name Your Price Event Tickets' extension through the 'Plugins' menu in WordPress at any time
